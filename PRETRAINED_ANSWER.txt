WHY COCO CLASSES ARE NOT DETECTING:
================================================================================

1. Training Data Issue:
   - We trained ONLY on FOD and tools (no COCO objects)
   - Model learned to output high scores for classes 80-85
   - COCO class outputs became suppressed

2. What happened during training:
   - Started: yolov8s.pt (80 COCO classes pretrained)
   - Extended: to 86 classes
   - Trained: on 10,671 images with ONLY classes 80-85
   - Result: COCO weights degraded (catastrophic forgetting)

3. Catastrophic Forgetting:
   - Fine-tuning on new classes without old class data
   - Model forgets previous knowledge
   - Common problem in transfer learning
